## Chapitre 1: Installation de l'environnement 💻 🛠 

<ol>
<li>
<b>Trello</b> : <br/>
Télécharger installer Trello pour iOS ou Android à l'adresse HTTPS://www.trello.com
<br/>
 Trello est une application de gestion des tâches selon le principe des to do list.
</li>
<li>
<b>Chocolatey</b>: <br/> Chocolatey et un utilitaire en ligne de commande permettant l'installation des paquets.
<br/>
https://chocolatey.org/install
</li>
<li>
 <b>Php 7</b>:<br/>
Php7 est la version de l'invite de commandes de l'interpréteur de code PHP que nous utiliserons durant cette formation.
<br/>
Télécharger et installer PHP à l'adresse suivante https://windows.php.net/downloads/releases/php-7.4.10-nts-Win32-vc15-x86.zip
 <br/> 
 Ajouter Php dans les variables d'environnement système afin qu'il soit accessible par tout utilitaire de ligne de commande. Pour se faire, suivez la procédure suivante 😏:
<ul>
<li>
Après avoir téléchargé l'une de versions de Php (la version 7 de préférence), lancer l'extraction dans un emplacement de l'ordinateur. 
</li>
<li>
Dans Rechercher, lancez une recherche et sélectionnez : Système (Panneau de configuration)
<li>
Cliquez sur le lien Paramètres système avancés.
</li>
<li>
Cliquez sur Variables d'environnement système 
</li>
Dans la fenêtre Modifier la variable système (ou Nouvelle variable système), indiquez la valeur de la variable d'environnement PATH, si vous êtes sur un système inférieur à Windows 10 vous devez ajouter un point virgule à la fin de la chaîne de valeur du Path, sur Windows 10 ajouter la valeur correspondant au chemin de l'exécutable du Php extrait plus haut. 
</li>
</ul> 
</li>

<li> <b>Composer</b>
<br/>
Composer un outil en commande line interface permettant la gestion des dépendances PHP.
<br/>
Suivez les instructions données par ce site Web 😏 https://davidperonne.com/installation-de-php-7-et-composer-sur-windows-10-en-mode-natif/
</li>
<li>
<b>Postgres SQL</b>
<br/>
Installez Postgres SQL après avoir téléchargé sur l'adresse https://sbp.enterprisedb.com/getfile.jsp?fileid=12715
</li>
<li>
<b>Visual Studio Code</b> 
<br/>
Visual Studio Code et un éditeur de texte très populaire doté d'une grande communauté, prétendre coder c'est sûr du c'est se munir des outils adéquats.
<br/>
Installer Visual Studio Code (VS Code) après son téléchargement à l'adresse https://aka.ms/win32-user-stable
</li>
<li>
<b>Github Extension for VS Code</b>
<br/> Ceci est une extension  de l'éditeur  qui intègre github  dans l'environnement de développement. <br/>
Si vous avez installé visual studio code sur votre pc, cliquez juste sur le lien suivant 😉 https://marketplace.visualstudio.com/items?itemName=KnisterPeter.vscode-github

<li>
<b>Compte Github</b> 
<br/>
Github est un outil de gestion des versions de code à distance. En gros c'est un hébergeur distant de codes source.
<br/>
Créez un compte github si vous n'en avez pas déjà sur l'adresse http://github.com/login
</li>
<li>
<b>Git</b> 
<br/>Git est un outil de gestion des versions de code en local. En gros il permet la manipulation du code source lorsqu'il est encore dans notre ordinateur local.
<br/>
Télécharger le à l'adresse http://git-scm.com
